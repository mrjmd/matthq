Service worker

October 5, 2019

I've got offline support working now so that once you've visited this site you can continue to browser and load the site even if you're offline. I thought about using a library like workbox for this purpose but wouldn't to stick to simplicity and writing from scratch for now. It's still mostly copypasta code but is quite simple and working well.

Service workers degrade gracefully so there is no adverse impact on older browsers. The biggest drawback with the current approach is it adds yet another manual step for the time being to each new post I create (assuming I'd like it cached). I now have to create and modify a new html file, add a teaser to the index.html file, and add an entry to my service worker's cache list variable each time. Mayba a little build system to handle some of this should be next...
